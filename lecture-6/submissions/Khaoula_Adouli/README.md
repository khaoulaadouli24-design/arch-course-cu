## PNG exports (diagrams)

The assignment asks for both `.drawio` and `.png` for the architecture and component diagrams. The `.drawio` files are included; you can export to PNG as follows:

1. Open `part2_api_architecture.drawio` or `part4_component_diagram.drawio` in [draw.io](https://app.diagrams.net/) (desktop or browser).
2. **File → Export as → PNG** (or use **Export** in the menu).
3. Save as `part2_api_architecture.png` or `part4_component_diagram.png` in this folder.

Alternatively, if your instructor accepts draw.io only, the `.drawio` files can be submitted as-is.
